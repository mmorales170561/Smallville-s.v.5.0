# 🛡️ Smallville Suite

An automated AI-powered bug bounty reconnaissance framework.

## 🚀 Cloud Deployment
This suite is designed to run for free on **Streamlit Community Cloud**.

1. Go to [share.streamlit.io](https://share.streamlit.app).
2. Connect this GitHub repository.
3. Set `app.py` as the main entry point.
4. **Deploy.**

## ⚡ Features
- **Observer:** Passive subdomain discovery.
- **Kingpin:** High-value target filtering.
- **Automated Hunt:** Full stealth-mode scanning.
